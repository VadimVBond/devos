# PROJECT_STATE.md

*   **Название проекта**: DevOS
*   **Назначение**: Оркестрационный слой (Kernel) для разработки с ИИ.
*   **Архитектура**:
    *   **Execution Layer (`ai/`)**: Детерминированный Python-код, рантайм для вызова LLM.
    *   **Cognitive Layer (`.ai/`)**: Декларативные данные, системные промпты, правила поведения.
    *   **Kernel Layer (`kernel/`)**: Оркестрация, планирование задач через Execution Graph (DAG).
*   **Стек**: Python 3.14+, Typer, SQLAlchemy 2.0 (Async), Pydantic, Loguru, SQLite.
*   **Основные функции**:
    *   **AI Graph Builder**: Генерация детерминированных графов задач (DAG) из естественного языка.
    *   **Execution State Manager**: Отслеживание жизненного цикла задач и управление зависимостями.
    *   **FS Plugin**: Инструменты для работы с файловой системой (mkdir, write, read, ls, exists).
    *   **Git Plugin**: Инструменты контроля версий (init, add, commit, status, branch).
*   **Текущий статус**: Реализованы базовые плагины `fs` и `git`. Ядро успешно выполняет сложные цепочки задач, включая инициализацию репозиториев и коммиты.
