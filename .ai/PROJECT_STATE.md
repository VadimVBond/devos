# PROJECT_STATE.md

*   **Название проекта**: DevOS
*   **Назначение**: Оркестрационный слой (Kernel) для разработки с ИИ. Детерминированное планирование и выполнение задач через плагины.
*   **Стек**: Python 3.14+, Typer, SQLAlchemy 2.0 (Async), Pydantic Settings, Loguru, SQLite.
*   **Структура проекта**:
    *   `kernel/`: Ядро (engine, planner, executor).
    *   `cli/`: Интерфейс командной строки.
    *   `memory/`: Хранилище (SQLite) и контекст.
    *   `plugins/`: Изолированные инструменты (git, fs).
    *   `.ai/`: Документация, промпты и AI Router.
*   **Основные функции**:
    *   Конвертация интента в JSON-граф задач.
    *   Валидация задач через политики безопасности.
    *   Выполнение через зарегистрированные плагины.
*   **Ограничения**: DevOS не выполняет действия напрямую; строгая stateless-модель ядра (состояние только в memory).
*   **Текущий статус**: Реструктуризация завершена, база данных инициализирована, CLI настроен. Ядро готово к реализации бизнес-логики.
